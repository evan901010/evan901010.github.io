============================================================

サンプルモデル　初音ミク  PRO版

============================================================


　初音ミクの原画を元に作成した Live2D モデルです。

　Cubism Editor 3.1の新機能『スキニング』でツインテールなどをなめらかに動かすことができます。
　物理演算をアニメーションに反映させる新機能『アニメーションベイク』も、すぐにお試しいただけるモデルです。

------------------------------
サンプルデータ構成
------------------------------

　モデルデータ(cmo3)　※物理演算設定を含む
　基本モーション(can3)
　組込用ファイル一式（runtimeフォルダ）
　・モデルデータ（moc3）
　・モーションデータ（motion3.json）
　・モデル設定ファイル（model3.json）
　・物理演算設定ファイル（physics3.json）
　・表示補助ファイル（cdi3.json）
